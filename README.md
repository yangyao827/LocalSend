# 🚀 LandDrop - 局域网文件互传小工具

一款基于 Python 实现的轻量级局域网文件/文本互传工具。默认端口 `8001`，支持 PC、手机、平板等多端设备互传。

## ✨ 主要功能
- **文件互传**：点击设备气泡即可发送多个文件。
- **文本传输**：长按设备气泡可发送剪贴板文本。
- **多端支持**：只要在同一局域网下，任何带浏览器的设备都能使用。
- **自动打包**：针对 iOS 优化，多个文件发送时自动打包成 ZIP，解决 Safari 下载限制。

## 🛠️ 使用方法

### 1. 启动服务
- **方式 A**：在控制台运行：`python land_drop.py`
- **方式 B**：编辑好 `start_lan_drop.vbs` 中的脚本路径，直接双击运行（后台静默运行）。

### 2. 访问地址
在浏览器中输入以下地址进入界面：
- **PC 本机**：`http://localhost:8001`
- **移动端/其他电脑**：`http://{电脑IP}:8001` 
  - *注：若 IP 经常变动，可尝试使用 `http://{电脑设备名称}.local:8001` 访问。*
- **首次登录**：输入任意设备名称即可加入网络。

### 3. 操作指南
- **发送文件**：直接 **点击** 目标设备的气泡。
- **发送文本**：**长按** 目标设备的气泡。
- **修改名称**：**长按** 代表自己的气泡（带“我”字样）。

## ⚙️ 进阶：开机自启动设置 (Windows)
如果你希望电脑开机后自动启动该服务，请按以下步骤操作：

1. 按下 `Win + R` 键，输入 `shell:startup` 并回车。
2. 在打开的文件夹中，将 `start_lan_drop.vbs` 脚本的快捷方式（或原文件）拖入即可。
